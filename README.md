# 🧮 Calculator App
A simple and functional Calculator application built with **React.js**. This project performs basic arithmetic operations with a clean and minimal interface.

## 📋 Project Overview
This is a frontend React project that demonstrates component-based architecture and state management by building a fully functional calculator with a classic button layout and real-time display.

## ✨ Features
- **Basic Operations** - Addition, Subtraction, Multiplication and Division
- **Clear Button** - Reset the calculator with C button
- **Decimal Support** - Perform calculations with decimal numbers
- **Real-time Display** - Instantly shows input and results on screen
- **Clean UI** - Simple and distraction-free classic calculator layout
- **React State Management** - Dynamically updates display without page reload
